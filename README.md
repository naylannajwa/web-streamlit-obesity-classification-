# ⚖️ Obesity Classification Web App

This repository contains a **Streamlit-based web application** designed to classify obesity levels using Machine Learning. It provides a simple and intuitive interface for users to input their health data and receive instant classification results.

## 🚀 Features

- **Interactive Dashboard**: Easy-to-use sliders and inputs for user data.
- **Real-time Prediction**: Instant classification based on a pre-trained ML model.
- **Data Visualization**: Insights and visualizations based on the obesity dataset.

## 🛠️ Tech Stack

- **Framework**: Streamlit
- **Language**: Python
- **Machine Learning**: Scikit-learn, Pandas, NumPy
- **Model**: Pre-trained classification model (`model.pkl`)

## 📂 File Structure

- `app.py`: The main Streamlit application script.
- `model.pkl`: The serialized machine learning model.
- `dataset.csv`: The dataset used for training and analysis.

## 🚀 Getting Started

1. Install the required libraries:
   ```bash
   pip install streamlit pandas scikit-learn
   ```
2. Run the application:
   ```bash
   streamlit run app.py
   ```
